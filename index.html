Biodata



Saya adalah seorang mahasiswa yang memiliki ketertarikan di bidang Front End Development dan juga Cyber Security, saya senang belajar akan hal baru dan memecahkan permasalahan yang sering saya hadapi ketika proses belajar. Untuk sekarang ini, saya baru menguasai pemrograman HTML ( HTML, CSS, JS ) dengan level pemula, saya juga mengusai sistem jaringan komputer ( Khususnya untuk sistem milik CISCO ) dengan tingkat pemula. Selain keterampilan diatas, saya juga tertarik dengan UI/UX dan Front End Mobile Developer.

Workshop & Kursus

Telah saya ikuti :

  • Konsultasi

    • Disini saya mendapatkan pengetahuan tentang keamanan jaringan.


  • Dasar DevOps

    • Disini saya mempelajari tentang pengetahuan dasar DevOps, mulai dari pengertian sampai apa saja jenis layanan yang bisa dimanfaatkan untuk pengembangan DevOps.

  • Wjangan Cisco

    • Di workshop ini, kaya belajar bagaimana membuat jaringan komputer untuk dua gedung yang berbeda. Mulai dari melakukan Subnetting, pengaturan IP Address, Pemilihan perangkat keras untuk membuat jaringan, belajar mengenai VLAN dan implementasinya serta ada juga pembelajaran tentang routing dan juga access control.


  • Pesan

    • Disini saya mempelajari tentang pengetahuan dasar Pemrograman Web, mulai dari struktur html yang baik dan juga mendapat pengetahuan tentang CSS.
